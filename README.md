# FleetDash

Real-Time Fleet Telemetry Dashboard


## Features

- Live Vehicle Tracking
- Real Time GPS Updates
- Vehicle Management
- Speed Analytics
- Live Alerts
- Interactive Map
- Socket.io Communication


## Tech Stack


Frontend:

- React
- Vite
- Leaflet
- Recharts
- Socket.io Client


Backend:

- Node.js
- Express.js
- MongoDB
- Socket.io
